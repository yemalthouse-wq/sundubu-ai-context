## P/L の分離ルール（絶対厳守）

### 本部P/L
**基準：BOX価格¥38,000（送料込）**

```
本部売上   = ¥38,000/BOX（送料込）
本部原価   = ¥15,170〜¥15,220（商品原価）+ ¥3,000（送料）= ¥18,170〜¥18,220
本部卸粗利 = 約¥19,780/BOX
```

### 店舗P/L（取引先側・デリバリー）
**基準：Uber売価¥1,460/食（基本スンドゥブ）**

```
Uber売上    = ¥1,460/食
Uber手数料  = 約30〜35%
店舗仕入れ  = ¥250（デリチャネル卸値・BOX価格と非連動）
店舗追加原価 = TBD（豆腐・卵・ネギ・アサリ・包材 各原価確定待ち）
```

### ❌ 混ぜてはいけないもの

- 本部P/LにUber手数料を含めない
- 本部P/Lに店舗追加原価（米・豆腐等）を含めない
- 店舗P/Lに本部の商品原価を含めない

**この2つのP/Lは絶対に混在させない。分析・レポート・シミュレーションすべてに適用。**
