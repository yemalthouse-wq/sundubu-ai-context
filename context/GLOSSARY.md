## P/L の分離ルール（絶対厳守）

### 本部P/L
**基準：BOX売価¥38,000（送料込）**

```
本部売上   = ¥38,000/BOX（送料込）
本部粗利   = ¥2,000/鍋 × 10鍋 = 約¥20,000/BOX（鍋エンジンモデル・粗利率57%）
本部原価   = 差引 約¥18,000/BOX（内訳は再確定待ち）
```

### 店舗P/L（取引先側）
**基準：Uber売価¥780/食**

```
Uber売上    = ¥780/食
Uber手数料  = 約¥234〜¥273（30〜35%）
店舗仕入れ  = ¥250（本部への卸値）
店舗追加原価 = ¥120（米・豆腐）
店舗粗利    = ¥780 − Uber手数料 − ¥370（店舗原価合計）
```

### ❌ 混ぜてはいけないもの

- 本部P/LにUber手数料を含めない
- 本部P/Lに店舗追加原価（米・豆腐等）を含めない
- 店舗P/Lに本部の商品原価を含めない

**この2つのP/Lは絶対に混在させない。分析・レポート・シミュレーションすべてに適用。**
