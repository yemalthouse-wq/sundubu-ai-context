## スンドゥブ田中家：卸販売モデル（Type 1）

### モデル定義

既存店舗の空き厨房を活用し、スンドゥブをUber Eats経由で販売する。
本部はBOX単位で卸し、取引先が調理・販売を担う。

```
本部（スンドゥブ田中家）
  → BOXを卸す
取引先（居酒屋・カフェ等）
  → 空き時間に仕込み・UberEatsで販売
エンドユーザー
  → Uber経由で注文・受け取り
```

### 対象業態（既存厨房活用）

- 居酒屋の昼（アイドルタイム活用）
- カフェの夜
- ラーメン屋の深夜

## P/L 分離ルール（絶対厳守）

### 本部P/L の計算式

```
本部売上   = ¥38,000/BOX（送料込）
本部原価   = ¥15,170〜¥15,220（商品原価）+ ¥3,000（送料）= ¥18,170〜¥18,220
本部卸粗利 = 約¥19,780/BOX
```

### 店舗P/L の計算式（取引先側・デリバリー）

```
Uber売上        = ¥1,460/食（基本スンドゥブ）
Uber手数料      = 約30〜35%
卸値            = ¥250（デリチャネル独立価格）
店舗追加原価    = TBD（豆腐・卵・ネギ・アサリ・包材 — 各原価確定待ち）
```

### ❌ 混ぜてはいけないもの

- 本部P/L に Uber手数料を含めない
- 本部P/L に店舗追加原価（米・豆腐等）を含めない
- 店舗P/L に本部の商品原価を含めない

**この2つのP/Lは絶対に混在させない。**

## Ye Malthouse：DX推進モデル

### モデル定義

個人事業主として運営を継続しながら、DXを段階的に導入する。
スンドゥブ田中家との横断コスト（AI-SEO・AI-Ops）は按分適用。

### 固定事項

| 項目 | 値 |
|---|---|
| 法人格 | 個人事業主継続（変更不可） |
| 横断適用部署 | AI-SEO部・AI-Ops部（コスト按分） |
