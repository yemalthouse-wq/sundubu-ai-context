## スンドゥブ田中家：卸販売モデル（Type 1）

### モデル定義

既存店舗の空き厨房を活用し、スンドゥブをUber Eats経由で販売する。
本部はBOX単位で卸し、取引先が調理・販売を担う。

```
本部（スンドゥブ田中家）
  → BOXを卸す
取引先（居酒屋・カフェ等）
  → 空き時間に仕込み・UberEatsで販売
エンドユーザー
  → Uber経由で注文・受け取り
```

### 対象業態（既存厨房活用）

- 居酒屋の昼（アイドルタイム活用）
- カフェの夜
- ラーメン屋の深夜

## P/L 分離ルール（絶対厳守）

### 本部P/L の計算式

```
本部売上   = ¥50,000/BOX
本部原価   = ¥20,301.8 + ¥2,000（送料）= ¥22,302
本部卸粗利 = ¥27,698/BOX
```

### 店舗P/L の計算式（取引先側）

```
Uber売上        = ¥780/食
卸値            = ¥250
店舗追加原価    = ¥120（豆腐¥35 + ご飯¥40 + 卵¥20 + 包材¥25）
1杯の店舗利益   = ¥410（¥780 − ¥250 − ¥120）
月間利益ターゲット = ¥369,000（¥410 × 30杯 × 30日）
```

### ❌ 混ぜてはいけないもの

- 本部P/L に Uber手数料を含めない
- 本部P/L に店舗追加原価（米・豆腐等）を含めない
- 店舗P/L に本部の商品原価を含めない

**この2つのP/Lは絶対に混在させない。**
