## 2026-03-10：卸販売P/L確定（→ 2026-03-13に価格改定）

### 1BOX構成（確定）

| 項目 | 数値 |
|---|---|
| 内容 | スンドゥブベース・豚キムチMIX・きのこMIX（3種セット） |
| 数量 | 10鍋分（1鍋=20食 → 200食分） |
| **BOX価格** | **¥38,000（送料込）** ← 旧¥50,000から改定 |
| 1鍋原価 | ¥1,517〜¥1,522 |
| 1BOX商品原価 | ¥15,170〜¥15,220 |
| 卸粗利 | 約¥19,780/BOX |

### 管理上の分離ルール（変更不可）

- **卸販売P/L**（本部側）= BOX売価 − BOX商品原価 − 送料
- **自店舗P/L**（取引先側）= Uber売上 − 仕入れ代 − 店舗側コスト
- **卸利益計算にUber手数料・店舗原価・包材費を混ぜない**
