# Shin-Maruko Pilot
Sundubu Tanaka OS

## Position
Pilot Store 0

## Objective
Verify whether the Lunch Insertion Model can generate repeatable profit in a real mixed urban node.

## Location Logic
Shin-Maruko = low fixed cost base
Musashi-Kosugi = high delivery demand zone

## Existing Format
- current brand: existing store brand
- Sundubu Tanaka used only for delivery
- dine-in remains under current brand

## Key Checks
- kitchen burner count
- fridge / freezer capacity
- idle lunch slot
- Uber potential
- operator willingness

## Expected Role
Not a final FC model.
This store is a pilot validation node.

## Timeline
- 開店目標: 2026年4月中
- 中間評価: 2026年7月（3ヶ月後）
- Final Go/No-Go: **2026年10月**（6ヶ月後）

## Go/No-Go Criteria（3軸）

### 軸1: 月次売上
| 判定 | 基準 |
|---|---|
| Go | ¥780,000以上/月（25杯/日） |
| 要観察 | ¥450,000〜779,999/月 |
| No-Go | ¥450,000未満/月（15杯/日未満） |

### 軸2: 客単価（AOV）
| 判定 | 基準 |
|---|---|
| Go | ¥1,300以上 |
| 要観察 | ¥1,100〜1,299 |
| No-Go | ¥1,100未満 |

### 軸3: リピート率
| 判定 | 基準 |
|---|---|
| Go | 35%以上 |
| 要観察 | 20〜34% |
| No-Go | 20%未満 |

### 総合判定
- 3軸Go → FC展開決定
