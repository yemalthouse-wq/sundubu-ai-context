## 1. 物件・内装

### 物件確認
- [ ] 物件の坪数確認（6〜20坪）
- [ ] 家賃確認（月売上の10%以内 / 上限¥300,000）
- [ ] 1階路面かどうか確認（Uberライダーアクセス）
- [ ] ガス管の有無確認
- [ ] 換気設備の確認
- [ ] 200V電源の有無確認
- [ ] 冷蔵・冷凍スペースの確認（30〜50食分の冷凍保管が可能か）
- [ ] デリバリー受渡口の設置可否確認（客導線と分離できるか）
- [ ] Uber Eats / Wolt の配達エリア内であることを確認

### 契約・内装
- [ ] 賃貸借契約締結
- [ ] 保証金・礼金の支払い
- [ ] 内装工事の要否判断（保守型は最小限）
- [ ] デリバリー受渡口の設置（必要な場合）
- [ ] 看板・表示の設置（[要確認]：既存店ブランドを維持するため不要の可能性）
