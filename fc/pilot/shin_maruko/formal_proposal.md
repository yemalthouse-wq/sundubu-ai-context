## はじめに

この資料は、スンドゥブ田中家の冷凍スープ・具材を貴店に卸販売し、
貴店の**空き時間の厨房**でデリバリー販売をしていただくためのご提案書です。

貴店の看板・既存のお客様には一切影響しません。
調理は**3分**。仕込みは**不要**です。

## 1. スンドゥブ田中家とは

**中目黒発のスンドゥブ（韓国風豆腐チゲ）専門のデリバリーブランドです。**

- 中目黒を拠点に、Uber Eatsで展開
- 「温かくて・早くて・一人前から頼める」をコンセプトに、ランチ・夜・深夜のデリバリー需要に対応
- スープ・具材をすべて本部で製造・冷凍パック化。加盟店は温めるだけで提供できる

### メニュー
- スンドゥブ（標準）
- チーズスンドゥブ
- 豚キムチスンドゥブ

### 新丸子エリアを選んだ理由
武蔵小杉のタワーマンション群（約1.2万世帯）まで自転車で3〜5分。
デリバリー需要が非常に高いエリアです。
近隣世帯のわずか**0.2%**が1日に注文するだけで、目標の25杯に達します。

## 2. 卸販売モデルのしくみ

```
スンドゥブ田中家（本部）
  ↓ 冷凍スープ・具材パックを週1回配送
貴店の厨房（空き時間に使用）
  ↓ 注文が入ったら3分で調理・盛り付け
Uber Eatsでお客様にお届け
```

### 貴店の作業は3ステップだけ

```
① 冷凍スープを湯煎または鍋で温める（約2分）
② 具材パックを加え、ひと煮立ち
③ 容器に盛り付けてUberへ渡す
```

**調理時間：約3分 / 1杯**
**1鍋で20杯分**が作れます。

### よくある心配ごと

| ご不安 | 実際のところ |
|---|---|
| 仕込みが大変では？ | **不要です。** 冷凍パックを温めるだけです |
| 厨房が汚れる・荒れる？ | **ほぼ影響なし。** コンロ・鍋・炊飯器のみ使用 |
| 在庫の置き場所は？ | **冷凍庫の一角で十分。** 約30〜50食分のスペースがあればOK |
| スタッフが忙しくなる？ | **隙間時間で対応可能。** 注文があったときだけ動く仕組みです |
| やめたくなったら？ | **いつでも停止できます。** 在庫を使い切れば終了 |

## 6. ブランド使用ルール

Uber Eats では「**スンドゥブ田中家**」の名前で出品していただきます。

### できること

- Uber Eatsの店舗名・メニュー名に「スンドゥブ田中家」を使用する
- 本部が提供する商品写真・説明文をそのまま使用する
- SNS・チラシで本部確認のうえ宣伝する

### できないこと

- レシピの独自アレンジ・原材料の変更
- ロゴ・写真の無断改変
- 契約終了後のブランド名の継続使用

### 品質基準

Uberのレビューが**3.5以下**になった場合、本部から改善のご連絡をします。
30日以内に改善が見られない場合、ブランド使用をいったん停止します。
（品質を守ることが、貴店とブランド双方の利益につながります）
