# 【提案】既存厨房の「空き時間」を月20万円の利益に変える
Sundubu Tanaka OS v0.3 / For Owner Pitch

## 1. 概要：何をやるか？
貴店のランチタイムまたはアイドルタイムを活用し、「スンドゥブ田中家」のメニューをデリバリー（Uber Eats 等）限定で販売します。
※既存の店名や看板を変える必要はありません。

## 2. 利益モデル：いくら儲かるか？
- **販売価格:** ¥1,200 (Uber Eats)
- **店舗受取:** **¥780** (手数料35%引後)
- **1杯の利益:** **¥280 〜 ¥320** (食材・光熱費込)
- **目標収益:** **月間 約20万円** (1日25杯 × 26日稼働の場合)
※加盟金50万円は、約2.5ヶ月で回収可能です。

## 3. 現場の負担：店主が気にする4つのこと
1. **仕込みはあるか？** → **NO.** OEMスープを温めるだけ。包丁はほぼ使いません。
2. **厨房は荒れるか？** → **NO.** レンジと炊飯器、コンロ2口あれば完結。現状のまま導入可能。
3. **在庫は嵩張るか？** → **NO.** 1箱(200食)単位の配送。冷凍庫の1/3程度のスペースでOK。
4. **忙しくなるか？** → **NO.** 1杯の調理時間は約3分。既存スタッフの隙間時間で回せます。

## 4. 導入スケジュール
- **Day 1:** 契約・システム登録
- **Day 3:** 食材納品・調理研修(30分)
- **Day 7:** 販売開始（Uber公開）

貴店の厨房が、何もしなくても月20万円を稼ぐ「収益エンジン」に変わります。
