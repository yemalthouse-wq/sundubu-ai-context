# 【個別提案】既存厨房の「空き時間」を月20万円の利益に変える
Sundubu Tanaka OS v0.3 / For Owner Pitch (Final)

## 1. 概要：何をやるか？
貴店のランチタイムまたはアイドルタイムの厨房を活用し、「スンドゥブ田中家」のメニューをデリバリー限定で販売します。
※既存の店名や看板、営業スタイルを変える必要は一切ありません。

## 2. 利益モデル：月間収益イメージ
- **店舗受取額:** **¥780** /杯 (Uber手数料引後)
- **平均利益:** **約 300 円** /杯 (原価・光熱費込)
  ※現場状況により ¥280〜¥320 の範囲で推移します。

**【収益の目安】**
- **通常ライン (25杯/日):** **月間 約 20 万円** の追加利益
- **最低ライン (10杯/日):** **月間 約 8 万円** の追加利益

**【投資回収】**
初期費用は約50万円。月20万円の収益により、**約3ヶ月で全額回収**が可能なモデルです。

## 3. 現場の負担：店主が気にする4つのこと
1. **仕込みはあるか？** → **NO.** 専用スープを温めるだけ。包丁は使いません。
2. **厨房は荒れるか？** → **NO.** レンジ、炊飯器、コンロ2口で完結。現状のまま導入可能。
3. **在庫は嵩張るか？** → **NO.** 1箱(200食)単位の配送。冷凍庫のわずかなスペースでOK。
4. **スタッフは忙しくなるか？** → **NO.** 調理時間は約3分。既存スタッフの隙間時間で回せます。

## 4. リスクについて
- **売れなかったら？** → デリバリー専用ブランドのため、既存営業への影響はゼロです。
- **撤退は可能か？** → いつでも停止可能です。大きな負債を抱えるリスクはありません。

## 5. 導入スケジュール
- **Day 1:** 契約・システム登録
- **Day 3:** 食材納品・調理研修(30分)
- **Day 7:** 販売開始（Uber公開）
