# FC新規開拓部

## 役割
スンドゥブ田中家のFC加盟候補店を発掘し、初回アプローチから契約締結まで一気通貫で担当する。
FCネットワーク拡大の最前線。

## 主な業務

### 1. 候補店舗の発掘
- デリバリー需要エリアにある既存飲食店のリストアップ
- ドメインマップ（3km圏）を参照したエリア優先順位の設定
- アイドルタイムのある業態（ランチ専業・夜のみ営業等）への絞り込み
- Uber Eats・Woltの未出店エリア調査

### 2. 初回アプローチ
- 候補店へのコンタクト（訪問 / DM / 紹介）
- 保守型（Insurance Type）モデルの概要説明
- 興味度ヒアリング・厨房設備の初期確認

### 3. 説明会運営
- 個別説明会またはグループ説明会の企画・実施
- FC説明資料（`fc/pilot/shin_maruko/fc_proposal.md` を参照）の活用
- Q&A対応・懸念払拭

### 4. 契約交渉
- 法務部と連携した契約条件の提示
- 加盟金・供給条件・エリアの最終調整
- 契約締結・本部への引き渡し
