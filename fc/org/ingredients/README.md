# 原材料発注部

## 役割
スープ・具材の原材料（豚肉・きのこ・キムチ等）を仕入れ先から調達する。
コスト・品質・安定供給の三角形を維持する。

## 主な業務
- 原材料の定期発注・仕入れ管理
- 仕入れ先との価格交渉・契約管理
- 原材料コストのモニタリングと本部報告
- 価格変動時の代替素材検討（レシピ改定部と連携）
- 食材の季節変動リスクの管理
