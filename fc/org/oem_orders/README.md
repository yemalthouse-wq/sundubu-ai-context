# OEM発注部

## 役割
スープベース・具材パック等のOEM製造を外部メーカーに発注・管理する。
品質基準を守りながら、コスト競争力のある調達を実現する。

## 主な業務
- OEMメーカーへの製造発注
- 製造仕様・品質基準の管理（レシピ改定部と連携）
- 納品検品・品質確認
- 価格交渉・契約更新
- 代替メーカーの調査・リスク分散
