# AI推進・業務自動化部

## 役割
Claude・ChatGPT・Gemini（AI三銃士）を中心に、社内業務をAIで自動化・効率化する。
受発注・レポート生成・在庫管理への AI連携を推進し、外部のAI活用企業との連携窓口も担う。

## 主な業務

### 業務自動化の設計・実装
- **受発注自動化**：加盟店/取引先からの発注をAIで受け付け・集計・出荷指示に変換
- **レポート自動生成**：売上・在庫・リピート率の月次レポートをAIで自動作成
- **在庫管理AI連携**：販売数データから発注推奨量を自動計算（鍋数ベース）
- **問い合わせ自動応答**：Uber/LINEからの定型問い合わせへのAI自動返信
- **ドキュメント管理**：このGitHubリポジトリをAIとの共有コンテキストとして運用

### 外部AI活用企業との連携
- AI系スタートアップ・開発会社との協業検討
- 飲食業向けAIツールの調査・評価・導入判断
