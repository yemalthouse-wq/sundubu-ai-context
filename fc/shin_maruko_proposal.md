# 【個別提案書】新丸子店限定：ランチ厨房収益化プロジェクト
Sundubu Tanaka OS v0.3 / 2026-03-09

## 1. 提案の趣旨
貴店の「昼の空き厨房」を、スンドゥブ田中家の「デリバリー拠点」として活用。
既存のオペレーションを妨げず、月間約20万円の追加利益を創出するプラグイン型モデルの導入提案です。

## 2. 収益シミュレーション（1杯あたりの経済）
店主の手元に残る現金を、最新の市場データに基づき算出しました。

| 項目 | 金額 | 備考 |
| :--- | :--- | :--- |
| **Uber販売価格** | **¥1,200** | 標準販売価格 |
| Uber手数料 (35%) | ▲¥420 | プラットフォーム利用料 |
| **店舗受取額** | **¥780** | **店主の手元に入る売上** |
| 食材原価・包材費 | ▲¥380 | スープ・具材・米・容器等 |
| **理論利益** | **¥400** | 1杯あたりの限界利益 |
| **現実利益** | **¥280〜¥320** | **広告費・光熱費等の増分を反映** |

## 3. 月間収益イメージ（現実ライン）
1日平均25杯（武蔵小杉エリアの標準的な需要）を販売した場合：

- **1日あたりの追加利益:** 約 ¥7,500
- **月間追加利益 (26日稼働):** **約 ¥195,000 〜 ¥210,000**
- **初期投資回収:** 加盟金50万円を、約2.5ヶ月で完全回収可能。

## 4. なぜ新丸子で「25杯」が確実なのか
新丸子拠点は、国内屈指のデリバリー需要エリアである**「武蔵小杉タワーマンション群（約1.2万世帯）」**まで自転車で3〜5分の好立地です。
- 近隣世帯のわずか **0.2%** が1日に注文するだけで25杯ラインを達成。
- スープ料理は保温性が高く、配送による品質低下（伸びる、冷める）が極めて少ないため、高リピート率が期待できます。

## 5. 導入条件
- **調理時間:** 約3分（仕込み不要、レンジ・炊飯器・コンロ使用）
- **稼働時間:** 11:00〜15:00（コアタイム）
- **必要設備:** 業務用電子レンジ、炊飯器、冷凍ストックスペース（30〜50食分）
